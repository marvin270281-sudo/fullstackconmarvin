Fullstack & Backend

Semana 11-13: Cerrando el ciclo entre el cliente y el servidor

Módulo 1: Node.js y Express

Entorno de ejecución y API frameworks robustos para servir datos al cliente.

  • Introducción a Node.js y Node Package Manager (NPM)
  • Creación de un servidor Express básico
  • Rutas, peticiones (req), respuestas (res) y Middlewares
  • Diseño de API RESTful, códigos de estado HTTP

Módulo 2: Bases de Datos (SQL & NoSQL)

Almacenamiento persistente, la fuente de la verdad para aplicaciones reales.

  • Modelado de datos básicos y relaciones
  • NoSQL: MongoDB con Mongoose (Schemas, Models)
  • SQL Básico: PostgreSQL / MySQL
  • CRUD Funciones: Create, Read, Update, Delete

Módulo 3: Autenticación, Seguridad y Deployment

Asegurando y lanzando tu aplicación completa a la web pública.

  • Manejo de JSON Web Tokens (JWT) y sesiones seguras localmente
  • Sanitización y validación segura de datos backend
  • Git profundo: Ramas, Pull Requests y resolución de conflictos
  • Cloud Deployment en plataformas (Vercel, Render o Netlify)
