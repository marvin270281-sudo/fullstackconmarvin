CSS3 & Diseño Responsivo

Semana 3-4: Estilizando y dando vida a la web

Módulo 1: Fundamentos de CSS

Entendiendo el cascada, especificidad y el modelo de caja, las bases fundamentales de todo CSS.

  • Selectores, clases, IDs y pseudo-clases
  • El Box Model (margins, borders, padding, content)
  • Posicionamiento (static, relative, absolute, fixed, sticky)
  • Variables CSS (Custom Properties)

Módulo 2: Layouts Modernos

Atrás quedaron los floats. Aprende a crear estructuras complejas fácilmente con Grid y Flexbox.

  • Flexbox: Alineación, dirección y distribución de espacio
  • CSS Grid Layout: Filas, columnas y áreas de template
  • Patrones comunes de UI con Flex y Grid
  • Media Queries para Responsive Design (Mobile First)

Módulo 3: Avanzado: Animaciones y Efectos

Dale el efecto "Wow" a tus aplicaciones web a través de transiciones fluidas y transformaciones.

  • Transiciones CSS y curvas bezier
  • Transformaciones 2D y 3D
  • Keyframes y animaciones complejas
  • Glassmorphism, sombras avanzadas (box-shadow) y gradientes
