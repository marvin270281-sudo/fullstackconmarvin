JavaScript Moderno

Semana 5-7: Domina la lógica de programación web

Módulo 1: Fundamentos y Lógica

Aprende las bases de la programación aplicadas al lenguaje de la web.

  • Variables (let, const), Tipos de Datos y Operadores
  • Estructuras de Control: if/else, switch, Bucles (for, while)
  • Funciones tradicionales y Arrow Functions (=>)
  • Arrays, Objetos y métodos básicos

Módulo 2: Manipulación del DOM y Eventos

Conecta la lógica con la interfaz gráfica para crear dinamismo.

  • Seleccionar y manipular elementos (querySelector, textContent)
  • Modificar estilos y clases usando CSSOM
  • Escuchar eventos (addEventListener, click, submit)
  • Burbujeo y delegación de eventos

Módulo 3: Javascript Avanzado (ES6+)

Entendiendo las bondades del JS moderno y asincronía.

  • Destructuring, Spread/Rest operators y Literales de Plantilla
  • Conceptos funcionales: map, filter, reduce
  • Asincronía: Callbacks, Promesas, y Async / Await
  • Peticiones HTTP: fetch() API y uso del formato JSON
