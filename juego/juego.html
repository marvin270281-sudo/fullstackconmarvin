¿Qué es JavaScript?

Es un lenguaje de programación usado para crear páginas web interactivas.

¿Qué es una variable?

Es un contenedor para almacenar datos que pueden cambiar durante la ejecución.

¿Qué es una función en JS?

Un bloque de código reutilizable que ejecuta una tarea específica al ser llamado.

¿Qué es el DOM?

Es la estructura de objetos (Modelo de Objetos del Documento) que representa una página web interactiva.

¿Qué hace addEventListener?

Escucha y asocia una función a un evento específico (como hacer clic o pulsar una tecla).

¿Qué significa HTML?

HyperText Markup Language. Proporciona la estructura semántica de cualquier sitio web.

¿Qué es CSS?

Cascading Style Sheets. Se utiliza para dar estilo, colores y diseño responsivo a la página web.

¿Qué es Flexbox?

Un modelo de diseño CSS optimizado para alinear y distribuir elementos en filas o columnas de forma flexible e inteligente.

¿Qué es el Box Model?

El 'Modelo de Caja' engloba todo elemento web: se compone de margenes (margin), bordes (border), relleno (padding) y contenido.

¿Qué es CSS Grid Layout?

Un potente sistema de maquetación 2D nativo en CSS basado en definir filas y columnas.

¿Qué es una Arrow Function?

Una sintaxis más corta (=>) introducida en ES6 para escribir funciones anónimas en JavaScript.

¿Para qué sirve map()?

Recorre un arreglo de datos y crea uno NUEVO con los resultados de la función aplicada a cada elemento.

¿Qué es una Promesa?

Un objeto que representa el fallo o éxito eventual de una operación asíncrona (ej. descargar datos de la base).

¿Qué es la API nativa fetch()?

Una función de JavaScript moderno usada pura realizar peticiones asíncronas HTTP a servidores y APIs REST.

¿Qué es React?

Una poderosa librería de JavaScript desarrollada por Facebook para construir interfaces de usuario basadas en componentes.

¿Qué es el Estado (State) en React?

Es la memoria interna de un componente. Al cambiar o actualizarse, hace que la interfaz se re-renderice automáticamente.

¿Qué es un Hook en React?

Funciones especiales (como useState o useEffect) que permiten "engancharse" a características vitales de los componentes funcionales.

¿Qué es el Virtual DOM?

Una copia ligera y en memoria del DOM real. React lo usa para calcular cambios y actualizar la web rápidamente sin recargar todo.

¿Qué son los 'props' en React?

Los atributos o datos pasados de componentes padres a componentes hijos. ¡Ojo! Son de solo lectura.

¿Qué es Node.js?

Un entorno de ejecución que permite procesar y usar JavaScript del lado del servidor (backend).

¿Qué es una API REST?

Un conjunto de reglas estandarizado que permite a diferentes aplicaciones comunicarse mediante métodos HTTP (GET, POST, PUT, DELETE).

¿Qué es SQL?

Structured Query Language. Un lenguaje diseñado específicamente para administrar y consultar bases de datos relacionales robustas.

¿Qué es Git?

El sistema de control de versiones distribuido más popular, vital para registrar los cambios en nuestro código y trabajar en equipo.

¿Qué es Express.js?

Un framework web rápido y minimalista para Node.js, frecuentemente utilizado para desarrollar APIs REST.

¿Qué es JWT?

JSON Web Token. Un estándar abierto usado en desarrollo backend para verificar la autenticidad y generar sesiones de login seguras.
